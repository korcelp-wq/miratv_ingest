Key Fix Implemented / Needed

Anchor-based extraction, not brace-based.

Anchor examples:

"episodes"

"seasons"

Extraction strategy:

Find anchor

Walk backward to nearest {

Balance braces from that point only

Extract JSON island

ConvertFrom-Json (best effort)

This accommodates:

huge arrays

unicode

ANSI noise

Newman verbosity

provider quirks
